드론배틀로얄

FPV 드론 레이싱 + 사격

수정탄 발사장치가 장착된 드론을
1인칭 시점(FPV)에서 조종하며
과녁을 맞추는 드론 스포츠입니다.

경기규칙

1. 완주

드론의 무사 귀환이
가장 중요한 첫 번째 임무입니다.

2. 임무 수행

타격에 성공한 과녁의 개수가
최종 승패를 결정짓는 핵심 요소입니다.

3. 랩타임

타격 점수가 동점일 경우,
더 빠르게 복귀한 팀이 우승을 차지합니다.

제품

드론배틀로얄 기체

  • 휠베이스 : 400mm
  • 프로펠러 : 7인치
  • 모터 : 2807 1300KV
  • 변속기 : 55A 4in1
  • 배터리 : 6S 4000mAh ~ 5200mAh

FUGA FC

  • 드론배틀로얄 공식 FC
  • 프로세서: STM32F405
  • 센서 : ICM-42688-P / BMP280
  • 펌웨어 : Ardupilot

과녁 시스템

  • OLED 점수 표시
  • 피격 시 부저 작동
  • 전원 : 보조배터리 Type-C 입력
